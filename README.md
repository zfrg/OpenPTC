# OpenPTC
## What-is
This project is based on Tencent EdgeOne Pages to build a dynamic service for obtaining official Phigros charts so that you can deploy your own version of Phitogether that can play the official Phigros charts.

## Usage
1. Fork this project.
2. Log in to the [Tencent Cloud Console](https://console.tencentcloud.com/).
3. Type "EdgeOne" in the search box and click to enter the EdgeOne product page.
4. Find "EdgeOne Pages" in the left navigation bar and click on it.
5. Click the "Add Site" or "New Application" button.
6. Link to your repo.
7. Use "Other" Framework.
8. Deploy and enjoy!

> [!IMPORTANT]
> Remember to replace [the link](https://github.com/zfrg/OpenPTC/blob/ef78e751b958d314d1894609b4774ff7a4018096/chapters.json#L9) before using in a production environment.
